\paragraph{Key design decisions.}
Three engineering decisions required deviation from the original repository:
\begin{enumerate}
    \item \textbf{Standalone inference script.}
          The original \texttt{train.py --mode inference} depends on a
          custom TFDS builder (\texttt{kvfrans/tfds\_builders}).
          A self-contained script was written to sample Gaussian noise
          directly, removing the dataset dependency.
    \item \textbf{No JIT compilation.}
          \texttt{@jax.jit} was disabled on CPU-only hardware due to
          out-of-memory (OOM, exit code 137) during XLA compilation of
          DiT-B. Eager mode was used instead.
    \item \textbf{Reduced Inception feature dimension.}
          \texttt{pytorch-fid} was run with \texttt{dims=192} instead of
          the standard $2048$ to reduce compute time on CPU.
          This produces \emph{relative} FID values that are not directly
          comparable to the absolute FID-50k reported by the authors.
\end{enumerate}

\paragraph{Reproducibility.}
All scripts, pinned package versions, and \texttt{fid\_results.json} are
available at \texttt{study-zone/} within the repository fork.
Random seed is fixed to \texttt{42} for all single-seed runs;
seed variance analysis uses seeds $\{42,\,123,\,2025\}$.
