# study-zone

Thư mục cô lập cho toàn bộ output thí nghiệm — không ảnh hưởng repo gốc.

```
checkpoints/   → pretrained & finetuned checkpoints (download từ Google Drive)
samples/       → generated images
fid_results/   → FID evaluation outputs
logs/          → training/eval logs
scripts/       → custom/modified scripts
notebooks/     → experiment notebooks
```

**Lưu ý:** Không commit các file lớn (checkpoints, samples) lên git.
